BrakeImage

JS Beta Lib that split a images i different part and you can add some feature, Clone the Repository and open the index file.

Have Fun.....

Required

You need JQUERY.

Config

var options;
options = {
    timeX: 20, /*Time of image*/
    img: '.br', /*Image Selecto could be ID or a CLass*/
    type: 'down', /*Time : line, up,down,sin*/
    gutter: 1, /* space between slices */
    position: 'none', /*Float: Right, Left or none*/
    factor: 3 /*Facto of separation in the top and bottom*/
};

var x = new BreakImage(options);
